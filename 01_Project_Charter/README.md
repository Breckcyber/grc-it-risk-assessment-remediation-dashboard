# Deliverable 1: Project Charter

This folder contains the Project Charter for the Technology Risk Assessment, Control Assurance and Remediation Program.

The charter defines:

- Engagement objective
- Business problem
- Scope and exclusions
- Assessment approach
- Risk methodology
- Stakeholders
- Governance
- Deliverables
- Assumptions
- Limitations
- Success measures

The charter establishes the boundaries, responsibilities, and expected outcomes of the engagement.
