# Deliverable 6: Control Testing and Evidence

This deliverable assesses the design and operating effectiveness of 15 technology and cybersecurity controls.

The work includes:

- Control testing register
- Evidence request log
- Risk-based sampling
- Individual control workpapers
- Exception analysis
- Findings documentation
- Effectiveness conclusions
- Management actions and retesting requirements
