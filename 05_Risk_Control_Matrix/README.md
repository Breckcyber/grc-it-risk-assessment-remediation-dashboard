# Deliverable 5: Risk and Control Matrix

This folder contains the Risk and Control Matrix developed for MapleCare Health Services.

The matrix links the 15 risks from the Risk Register to:

* Existing controls
* Control gaps
* Recommended controls
* Control owners
* Control frequency
* Control type and nature
* Testing procedures
* Evidence requirements
* NIST CSF 2.0 references
* NIST SP 800-53 references
* ISO/IEC 27001:2022 references

The matrix was used as the foundation for the control-testing activities completed in Deliverable 6.
